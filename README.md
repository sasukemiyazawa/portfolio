# Portfolio SPA

Vite + React で構築した **シングルページアプリケーション（SPA）** のポートフォリオサイトです。  
バックエンドを持たず、フロントエンドのみで完結する軽量構成。  
UI は **MUI（Material UI）** をベースに、シンプルで見やすいデザインを採用しています。

こちらでデプロイしています。

https://sasukemiyazawa.github.io/portfolio/


---

## Features
- **SPA構成**：ページ遷移が高速でストレスのない操作感  
- **MUIベースのUI**：統一感のあるデザインとレスポンシブ対応  
- **Viteビルド**：高速起動・高速ホットリロード  
- **バックエンドなし**：静的ホスティング（GitHub Pages など）に最適  

---

## Tech Stack
- **Vite**
- **React**
- **MUI (Material UI)**
- TypeScript
- React Router
